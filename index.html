Principais tags html e seletores css

Tags basicas do HTML:

A tag HTML é usada para inicio e fim de um documento em html, HTML é uma marca que define um conteúdo de documento das respectivas linguagem, Ela envolve textos e outros elementos.

A tag BODY

A tag BODY define um corpo de um site, È na tag BODY que se encotra as informaçoes visiveis para o usuário.

Tag < p >

A tag < p > é usada para indicar um paragrafo

Tag < pre >

A tag < pre > é a tag utilizada para representar textos pré formatado. O texto dentro dessa tag é exibido em uma fonte não proporcional da mesma maneira em que o texto original foi disposto no arquivo. A tag < pre > desabilita a quebra de linha, desmarca o paragrafo entre outras coisas.

Tag < h1...h6 >

As tag < h1...h6 > são conhecidas como etiquetas de titulos e subtitulos, Elas ajuda organizar seu conteúdo.

Tag < span >

A tag < span > tem a funcionalidade agrupar/ou delimitar elementos a nível de linha.

Tag < div >

A tag < div > Cria um bloco genérico para agrupar conteúdos, Sua função é separar conteúdos de codigos HTML.

[SELETORES] 
            É um padrão de elementos e outros termos quen informa um navegador quais elementos HTML devem ser selecionados para que os valores 
          de propriedade CSS  dentro da regra seja aplicado a eles.
          
          
SELETOR POR TAG: É o seletor mais abrangente da CSS, Ele casa com todos os elementos encontrados.

SELETOR POR CLASSE:  O seletor por classe pode ter um ou mais itens definidos em seu atributo de  classe.
                    seletor classe pode corresponder a qualquer elemento que tenha classe aplicada a ele.

SELETOR POR ID: É usado para quando desejamos indentificar um elemento específico no seu documento, no CSS
                é usado "#".
